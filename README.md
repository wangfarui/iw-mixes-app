# 项目说明

## 配置说明

1. 复制 `config/index.example.js` 为 `config/index.js`
2. 在 `config/index.js` 中填写实际的小程序 appid
3. 确保 `config/index.js` 和 `manifest.json` 不会被提交到代码仓库
4. 运行 `npm run update-manifest` 更新 manifest.json 中的 appid
